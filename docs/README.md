# MONSTOR — Motion Charts

This folder contains the production-ready site served by GitHub Pages.

Files:
- `index.html` — main site
- `styles.css` — styling
- `script.js` — chart logic and interactions
- `favicon.svg` — site favicon

To preview locally:

```bash
# start a simple static server from the docs/ folder
cd docs
python3 -m http.server 8000
# then open http://localhost:8000
```
